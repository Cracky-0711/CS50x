How to use this tool

This is a web application to manage tasks. You can add, edit, delete tasks and magage deadlines.

Here is how to use this web application:

  1. Register an account from the Login menu in the header menu.


  2. Log in with your registered user name and password, and go to the page of task list.
  3. Click on "Add task" to add a new task.
  4. You can filter the tasks displayed by using the checkboxes in the "Display" section.
  5. You can edit the status and deadline of a task, or delete a task by clicking on the hamburger button to the right of each task.

  6. If you log out, you can hide the task list until you log in again.
